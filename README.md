### Название телеграмм бота: LINKIN

![img.png](logo.jpeg)

### Описание бота:

Бот предназначен для знакомств, поиска друзей, деловых партнеров. Пользователь создаёт анкету, получает
подборку подходящих профилей и может ставить лайки и дизлайки. При взаимном мэтче бот  отправляет контакт.
